# How-To Guides

Practical recipes for common pico-server-auth tasks.

## Guides

- [Embed in App](embed-in-app.md) — Run pico-server-auth inside your pico-boot application alongside your own controllers.
- [Standalone](standalone.md) — Deploy pico-server-auth as a dedicated auth microservice.
- [Custom Challenge Store](custom-challenge-store.md) — Replace the in-memory challenge store with Redis, a database, or any shared storage.
- [Testing](testing.md) — Test your application with pico-server-auth in pytest.
