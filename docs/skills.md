# AI Coding Skills

Install pico skills for AI-assisted development:

```bash
curl -sL https://raw.githubusercontent.com/dperezcabrera/pico-skills/main/install.sh | bash
```

| Command | Description |
|---------|-------------|
| `/add-component` | Add components, factories, interceptors, settings |
| `/add-tests` | Generate tests for pico components |
| `/init-project` | Scaffold a new pico-boot project |

See [pico-skills](https://github.com/dperezcabrera/pico-skills) for details.
